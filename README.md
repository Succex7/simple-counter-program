# Counter App

A simple and responsive counter application built with **HTML**, **CSS**, and **JavaScript**.  
It allows users to increase, decrease, and reset a numeric value using interactive buttons.


## Features
- Increase the counter value by 1
- Decrease the counter value by 1
- Reset the counter to 0
- Fully responsive layout
- Instant UI updates using JavaScript DOM manipulation

## Technologies Used
- **HTML5** — structure
- **CSS3** — styling & responsiveness
- **JavaScript (Vanilla)** — interactivity and logic


## How It Works
The application listens for button click events and updates the displayed number dynamically without reloading the page.

| Button | Action |
|------|------|
| Increase | Increments the value |
| Decrease | Decrements the value |
| Reset | Returns the value to zero |

## Getting Started
1. Clone or download the repository
2. Open `index.html` in your browser

No installation or dependencies required.

##  Purpose
This project demonstrates basic DOM manipulation, event handling, and responsive UI design using pure JavaScript without any frameworks.
